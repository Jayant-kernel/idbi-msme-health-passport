# MSME Financial Health Passport

We can switch between three MSME personas and toggle GST, UPI, EPFO, Account Aggregator, and Receivables/TReDS sources.
Each toggle recomputes the score, confidence, reason codes, and recommended loan range without crashing on partial data.
The prototype demonstrates an explainable, viability-based Track 03 Financial Health Card for IDBI Innovate 2026.
